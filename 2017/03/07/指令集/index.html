英特尔/AMD

简单指令集：手机

​ 32位： i386

​ 64位：armv7 arm7s armv10 mv10

安装nodejs（即是node）

nvm 安装nodejs

全称：node version manager

建议安装 node-v6.9.5-x64.exe 稳定版 （6.9以上最好）

开启：在命令面板中输入node

关闭：ctrl+C 两次 或者 .exit

配置系统环境变量：

一般Windows的启动文件在bin目录中，意思是启动目录，如果将开启目录配置到系统环境变量中，在命令控制面板中，直接输入名称，即可直接启动，
好处：方便我们在终端快启动我们的软件，有点类似于桌面快捷方式

示例：

1、复制路径：C:\Program Files (x86)\Tencent\QQ\Bin
2、添加到系统环境变量中

img:arrow_down_small:

3、在cmd中输入QQ，即可启动QQ

误删快捷方式：

1、在系统路径中找到的快捷方式打开（发送到桌面）
2、配置系统环境变量，用终端打开
